# AWS OpenSearch
amazon opensearch cli의 기능을 웹으로 구현 

## Features
- 인덱스 생성, 삭제, 별칭 지정
- bulk 인덱싱, upsert 인덱싱
- 키워드 자동 완성
- 검색
